This project uses Ajax technology to receive a list of users and display them in a table. You can edit, delete, or search or sort any record in the table.

for execute this app :
move to: C:\wamp64\www\ajax
All file in "ajax" folder move to "ajax" folder in "C:\wamp64\www\ajax" address
open wamp webserver and enter "http://localhost/ajax/index.php/" in browser 
open MySQL database on wamp (default username and password -> username: root, password: nothing)
open MySQL database on wamp (current username and password -> username: admin, password: admin)

if not found database file you can import "ajax.sql" in phpMyAdmin
